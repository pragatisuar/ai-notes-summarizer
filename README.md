# AI Study Notes Processor

A simple AI-inspired web app that converts long study notes into:
- Concise summaries
- Key keywords
- Exam-oriented questions

🔗 [Live App](https://ai-notes-summarizer-je9xy8as4iqbkuxinrzse8.streamlit.app/)

---

## 🚀 Features
- Summarizes long text into short, readable points  
- Extracts important keywords using frequency analysis  
- Generates possible exam questions  
- Download results as a text file  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  

---

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
